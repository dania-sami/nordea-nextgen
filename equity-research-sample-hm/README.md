# Equity Research Sample – H&M (Hennes & Mauritz)

This is a simple sample equity research project created for a Nordea Equity Research summer internship application.

It includes:
- A one-page style research note on H&M
- A basic valuation file with simple metrics (P/E, growth, margins)
- Example data used for the analysis

## Files

- `research_note_HM.md` – Main one-page research note
- `valuation_HM.xlsx` – Simple valuation model
- `data/hm_sample_data.csv` – Example financial figures
- `README.md` – This file

## How to use

You can open the valuation file in Excel and the note files in any text editor or preview them directly on GitHub.
